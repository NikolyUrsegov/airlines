## How to install
Make sure Git, Node.js is installed.

    # Use Git and clone project from github
    git clone https://github.com/NikolyUrsegov/airlines.git
    
    # Go to the project folder
    cd airlines
    
    # Install dependencies
    yarn install
    
## How to run

    # Start the development server
    yarn start
